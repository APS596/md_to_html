# Introduction

mdhtml is a C++ program which turns markdown source into html files

# Compilation

Clone the repository
>git clone https://github.com/APS596/md\_to\_html.git

Run the makefile (uses the GCC compiler)
>make

for other compilers, shouldn't be to hard to compile a single file with no external dependencies

# Usage

> mdhtml source.md > page.html

# Included features (work in progress)

Markdown syntax features already implemented : 
- titles (level 1 to 6) using #
- \**italic*\*
- \***bold**\*
- \****bold-italic***\*
